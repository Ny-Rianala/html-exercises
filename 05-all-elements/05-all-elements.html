All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The abbr element (<abbr>) represents an abbreviation or acronym.

<article>

The article element (<article>) represents a section of a page that consists of a composition that forms an independant part of document , page or site.

<aside>

The aside element (<aside>) represents a section of a page consisting of cotent that is related to the content around the aside element.

<b>

The b element (<b>) makes the font thicker.

<br>

The br element (<br>) represents a line break.

<body>

The body element (<body>) represents the main content of document.

<button>

The button element (<button>) represents a button.

<dd>

The dd element (<dd>) represents description, definition, or value, part of term-description group in a decription list.

<div>

The div element (<div>) represents its children, it can be used with a class, lang and title attributes to mark up semantics common to a group of consecutive element.

<dl>

The dl element (<dl>) represents a description list consisting of zero or more name-value groups.

<dt>

The dt element (<dt>) represents the term, or name, part of a term-description group in a description list.

<footer>

The footer element (<footer>) represents the footer of a document, or section of document.

<h1-h6>

The h1-h6 elements (<h1-h6>) represent headings and subheadings.

<head>

The head element (<head>) represents a collection of metadata for a the document.

<header>

The header element (<header>) represents the 'header' of the document or a section of document.

<html>

The html element (<html>) represents the root of an html document.

<iframe>

The iframe element (<iframe>) represents a nested browsing context.

<img>

The img element (<img>) represents an image.

<nav>

The nav element (<nav>) represents navigation of a document.

<ol>

The ol element (<ol>) represents a list of items, where the items have been intentionally ordered.

<script>

The script element (<script>) allows authors to include dynamic script and data blocks in their documents.

<section>

The section (<section>) represents a generic document or application section.

<style>

The style (<style>) allows authors to enbed style information in their documents.

<table>

The table (<table>) represents data with more than one dimension, in the form of a table.

<tbody>

The tbody (<tbody>) represents a block of rows that consist of a body of data for the parent table element.

<td>

The td element(<td>) represents a data cell in a table.

<th>

The th element(<th>) represents a header cell in a table.

<title>

The title element(<title>) represents the title or the name of the document.

<tr>

The tr element(<tr>) represents a row of the cells in a table.

<ul>

The ul element(<ul>) represents a list of items, where the order of the items is not important.
